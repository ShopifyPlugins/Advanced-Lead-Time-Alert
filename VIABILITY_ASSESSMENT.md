# BizBot 副业可行性评估（改造后）

> 评估日期: 2026-02-11
> 前身: EnergyPulse (欧洲电价预测 Bot) → BizBot (白标 AI 客服 Bot)

## 项目概述

BizBot 是一个面向小企业的白标 AI 客服 Telegram Bot。企业通过 Telegram 命令上传 FAQ/产品文档，Bot 基于 TF-IDF RAG 检索 + LLM 自动回答客户问题，支持人工升级和对话记录。

## 与 EnergyPulse 的关键差异

| 维度 | EnergyPulse (旧) | BizBot (新) |
|------|------------------|-------------|
| 目标客户 | C端消费者 | B端小企业 |
| 定价空间 | 3-5 EUR/月 | $50-200/月 |
| 到$1K/月需要 | ~5000免费用户+转化 | 5-10个付费客户 |
| 市场规模 | HEMS $48亿 | AI客服机器人 $93亿 (2025) |
| 市场增速 | CAGR 16% | CAGR 23.3% |

## 市场背景

- AI 客服聊天机器人市场 2025 年 $93 亿，2030 年预计 $272 亿（CAGR 23.3%）
- 客服场景占聊天机器人市场的 41.8%（最大应用领域）
- 75% 的中小企业正在尝试聊天机器人，SMB 细分 CAGR 高达 25.1%
- 聊天机器人每次交互成本 $0.50-0.70 vs 人工客服 $8-15

## 有利因素

1. **B2B 定价空间** — 竞品定价 $40-500/月，$50-200 完全合理，对比旧项目 3-5 EUR/月提升 10-40 倍
2. **已验证的独立开发者路径** — Chatbase 从 0 到 $1M ARR 用 5 个月；SiteGPT $500 投入，1 个月达 $10K MRR
3. **MVP 已具备核心功能** — 知识库管理 + RAG 检索 + LLM 生成 + 对话记忆 + 人工升级，~600 行代码
4. **获客渠道多** — Product Hunt / SEO / 直销本地商户 / Building in Public
5. **ROI 易论证** — 企业每月花 $100 省下一个兼职客服，决策门槛极低

## 主要风险

1. **赛道拥挤** — Chatbase、Botpress、Tidio、ManyChat 等已有数年积累和多渠道支持
2. **仅限 Telegram** — 大多数小企业客户通过网站或 WhatsApp 联系客服，非 Telegram
3. **AI 聊天机器人流失率高** — 行业月流失 6-12%（年流失 53-76%），远高于 SaaS 平均水平
4. **功能差距** — 缺少网页嵌入 widget、文件上传、可视化管理后台
5. **LLM API 成本** — 随规模线性增长，约 $15/月 (10客户 x 50条/天)

## 下一步行动

1. **网页嵌入式 widget** — FastAPI 端点 + JS snippet，打开网站客服市场
2. **PDF/TXT 文件上传** — 降低知识库 onboarding 摩擦
3. **找 1-2 家本地小企业免费试用** — 验证价值，获取案例
4. **Product Hunt 发布** — 准备 landing page 后争取首日爆发流量
5. **用量计费模式** — 数据显示比固定月费降低 46% 流失率

## 结论

从 3 星提升到 4 星。B2B 定位 + 更高定价空间 + 更短变现路径，让项目从"可以试试"变成"值得认真做"。最大瓶颈不再是变现能力，而是在拥挤赛道中的差异化——加上网页 widget 和 WhatsApp 支持后才真正具备与 Chatbase/Tidio 竞争的能力。
